# Assessment

## Project Overiew

#### This assessment focuses on implementing a stock management system using Java. It includes Java classes, polymorphism, GUI interfaces, test cases, and UML diagrams, all of which demonstrate the calculations for adding and selling stock, updating prices, and applying VAT. The project is completed in structured steps, using a main class and multiple subclasses to ensure each component functions correctly and displays accurate results through systematic testing. The system models stock items for an accessories shop and demonstrates key OO concepts including inheritance, polymorphism, method overriding, encapsulation, and dynamic binding.



## Implemented subclasses:
#### NavSys – GeoVision Sat Nav navigation system/ Jumper – Clothing item with size and material/ Jewellery – Jewellery item with type and weight/ PhoneCase – Phone case with model and colour
